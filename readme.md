# landis

A simple out-of-band Minecraft world map renderer and leaflet-based frontend

### usage

- create an `out` dir
- put a minecraft server at `./mcserver`
- `cargo run`
- let world save/generate, etc
- visit http://localhost:3090/
- refresh for new tiles
